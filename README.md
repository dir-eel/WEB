# DIREEL — Sitio Web Oficial

Repositorio del sitio web oficial de **DIREEL**, empresa dedicada al diagnóstico, reparación y mantenimiento de tractocamiones diésel.

## Descripción

Este proyecto contiene la página web institucional de DIREEL, desarrollada para presentar los servicios, especialidades, información de contacto y presencia digital de la empresa.

El sitio está enfocado en servicios para unidades de servicio pesado, diagnóstico diésel, mantenimiento, reparación y atención técnica especializada.

## Tecnologías utilizadas

* React
* JavaScript
* HTML
* CSS
* GitHub Pages
